# Junaid Irfan Portfolio

Professional portfolio for Junaid Irfan, a Full Stack MERN Developer based in Lahore, Pakistan.

## Live
https://junaid-portfolio-two-phi.vercel.app

## Stack
React.js, Vite, Tailwind CSS, JavaScript, HTML, CSS

## Includes
- About, Skills, Projects, Experience, and Education pages
- Light and dark mode
- Responsive design for desktop and mobile
- Project screenshots, live links, and source links
- Resume download and contact links

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
